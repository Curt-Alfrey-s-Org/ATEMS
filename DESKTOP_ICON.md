# ATEMS Desktop Icon

## Add to your Desktop

1. Copy `ATEMS.desktop` to your Desktop:
   ```bash
   cp /home/n4s1/rankings-bot/ATEMS/ATEMS.desktop ~/Desktop/
   ```

2. Allow launching (Linux Mint / Nautilus):
   - Right-click the icon on your Desktop
   - Choose **Allow Launching** (or **Trust and Launch**)

3. Double-click **ATEMS** to start the server.

A terminal will open, the server will start at http://127.0.0.1:5000, and your browser will open automatically after a few seconds. Press Ctrl+C in the terminal to stop the server.
